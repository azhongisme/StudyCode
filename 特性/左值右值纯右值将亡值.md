```cpp
int a = 1;
decltype(a); //int
decltype((a)); //int &
```
对于第一个比较好理解，第二个的话是decltype一个表达式，如果a是一个纯右值那么就是int,但a是一个左值，因此是一个引用
# std::move
```cpp
int a = 1;
decltype(std::move(a)); //int &&
```
将一个左值变为右值引用

```cpp
void func(int && a) {
  printf(__PRETTY_FUNCTION__);
}

void func(int a) {
  printf(__PRETTY_FUNCTION__);
}
```
对于这种函数重载会发生错误，因为当调用func(1)的时候不止一个选项。prvalue和xvalue无法进行重载
```cpp
void func(int && a) {
  printf(__PRETTY_FUNCTION__);
}

void func(int &a) {
  printf(__PRETTY_FUNCTION__);
}
```
可以进行这种修改
```cpp
void func(int && a) {
  printf(__PRETTY_FUNCTION__);
  std::cout<<std::endl;
}

void func(int &a) {
  printf(__PRETTY_FUNCTION__);
  std::cout<<std::endl;
}

int main() {
  func(1);//prvalue
  int a = 2;
  func(a);//xvalue
  func(std::move(a));//lvalue
}
```
